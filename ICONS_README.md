# PWA İkonları

Bu klasörde PWA için gerekli ikonlar bulunmaktadır. Şu anda SVG placeholder ikonlar mevcuttur.

## Gereksinimler

PWA'nın tam işlevselliği için PNG formatında ikonlar önerilir:
- `icon-192.png` - 192x192 piksel
- `icon-512.png` - 512x512 piksel

## İkon Oluşturma

1. SVG dosyalarını PNG'ye dönüştürebilirsiniz
2. Veya profesyonel bir tasarım aracı kullanarak yeni ikonlar oluşturabilirsiniz
3. İkonlar GlucoPilot temasını yansıtmalıdır (mavi tonlar, #3367D6)

## Geçici Çözüm

Şu anda SVG ikonlar kullanılmaktadır ve çoğu tarayıcı bunları destekler. Ancak bazı platformlarda PNG ikonlar daha iyi görünebilir.
